# Onboarding_v1 — Merchant Onboarding Application

Mobile-first merchant onboarding flow with:
- 3-document upload (Photo ID, Voided Check/Letter, W-9)
- Optional OCR prefill (consent-gated)
- PDF generation + merge order: Application → Photo ID → Check/Letter → W-9
- Email notifications (internal + merchant)
- Success page with PDF download
- Google Drive upload + WhatsApp notify + Monday.com item creation
